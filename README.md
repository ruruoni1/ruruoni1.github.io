# ruruoni1.github.io

GitHub Pages용 정적 랜딩 페이지입니다. 현재는 `GoPoint`와 `GoVerse82` 채널을 소개하는 1페이지 허브로 구성되어 있습니다.

## Files

- `index.html`: 메인 페이지 구조
- `assets/styles.css`: 레이아웃, 색상, 반응형 스타일
- `assets/site-data.js`: 프로젝트와 채널 데이터
- `assets/main.js`: 카드 렌더링 스크립트

## How to update

프로젝트를 추가할 때는 `assets/site-data.js`의 `projects` 배열에 항목을 추가하면 됩니다.

## Publish

1. GitHub에서 `ruruoni1.github.io` 저장소를 생성합니다.
2. 이 폴더 내용을 저장소 루트에 올립니다.
3. 기본 브랜치에 push 하면 GitHub Pages 루트 도메인으로 배포됩니다.
