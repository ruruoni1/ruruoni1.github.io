채널에서 다루는 것

    이 페이지 성격

    템플릿용 예시 사이트가 아니라, 제가 만든 것들을 직접 정리하고 연결해 두는 실제 운영 페이지로 구성했습니다.

    이 페이지

    이 사이트를 왜 따로 운영하느냐면

    GitHub 저장소만으로는 프로그램의 용도와 맥락이 잘 안 보일 때가 많아서, 고버스TV에서 다루는 내용과 함께 제가 만든 도구를 따로 정리해 두는 허브가 필요했습니다.

    이 페이지는

    필요해서 만든 도구를 남겨두는 곳

    영상 제작, 화면 공유, 테스트 과정에서 직접 불편했던 점을 해결하려고 만든 프로그램들을 모아 두고, 무엇을 하는 도구인지 짧고 분명하게 설명하려고 합니다.

    운영 방식

    직접 써보면서 계속 다듬는 방식

    먼저 제가 직접 사용해 보고, 쓸 만하다고 판단한 것만 소개 페이지와 저장소, 다운로드 링크까지 연결해 두는 방식으로 운영할 생각입니다.

    앞으로

    항목은 많지 않아도 하나씩 제대로

    지금은 GoPoint부터 정리해 두었고, 다음 프로그램도 준비가 되면 같은 방식으로 추가해서 개인 작업 기록이 쌓이는 페이지로 키워가려고 합니다.

    프로그램

    지금 소개 중인 프로그램

    현재는 GoPoint 하나를 먼저 정리해 두었습니다. 프로그램 수를 급하게 늘리기보다, 공개한 항목마다 소개 페이지와 저장소 링크를 제대로 연결해 두는 쪽으로 운영하고 있습니다.
